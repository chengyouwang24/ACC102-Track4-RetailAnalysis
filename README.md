# ACC102 Track4 - Merchant Full Operation Analytics System
## Interactive Data Analysis Tool (Local Only)

### 1. Project Overview
This project is a local-only interactive retail data analysis dashboard built with Python and Streamlit.
It provides full-link business analysis including sales, profit, category health, discount efficiency, return risk, inventory planning, and data-driven business strategies.

### 2. Core Functions
- Business Overview & KPI Dashboard
- Product Category Health Score (0-100)
- Monthly Growth & Seasonality Analysis
- Discount Profit & Break-Even Analysis
- Return Risk Control & Industry Benchmark
- Intelligent Inventory Suggestion
- Automatic Business Strategy & Report Export

### 3. Required Files
The following files must be placed in the same folder:
- app.py                 Main program file
- business.retailsales.csv     Detailed transaction data
- business.retailsales2.csv    Monthly summary data
- requirements.txt        Python dependency list
- README.md              Project description (this file)

### 4. How to Run Locally
1. Install required packages:
pip install -r requirements.txt

2. Launch the dashboard:
streamlit run app.py
